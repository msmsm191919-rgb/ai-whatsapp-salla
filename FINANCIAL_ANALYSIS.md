## 🔌 1. تكلفة OpenAI (الذكاء الاصطناعي)

### الأسعار الرسمية (مايو 2026):

| الموديل | Input | Output |
|---------|-------|--------|
| **GPT-4o-mini** (الأساسية) | $0.15 / مليون token | $0.60 / مليون token |
| **GPT-4o** (النمو/الشركات) | $2.50 / مليون token | $10.00 / مليون token |

### حساب الـ tokens لكل رسالة:
- متوسط رد العميل: **500 input + 150 output token** (System prompt + history + الرسالة الواردة)

### التكلفة الفعلية:

**الأساسية (10,000 رسالة شهرياً، gpt-4o-mini):**
```
Input:  10,000 × 500 = 5M tokens × $0.15  = $0.75
Output: 10,000 × 150 = 1.5M tokens × $0.60 = $0.90
─────────────────────────────────────────────────
المجموع: $1.65 / شهر ≈ 6.2 ر.س
```

**النمو (افتراض 30,000 رسالة، gpt-4o):** ⚠️
```
Input:  30,000 × 500 = 15M tokens × $2.50  = $37.50
Output: 30,000 × 150 = 4.5M tokens × $10.00 = $45.00
─────────────────────────────────────────────────
المجموع: $82.50 / شهر ≈ 309 ر.س ← أعلى من السعر!
```

**النمو (افتراض 5,000 رسالة realistic):**
```
Input:  5,000 × 500 = 2.5M × $2.50 = $6.25
Output: 5,000 × 150 = 0.75M × $10  = $7.50
─────────────────────────────────────────────────
المجموع: $13.75 / شهر ≈ 51.5 ر.س
```

## 📱 2. تكلفة WhatsApp Cloud API (Meta)

### نظام تسعير Meta — حسب **المحادثة** لا الرسالة:
> محادثة = نافذة 24 ساعة مع نفس العميل

**أسعار السعودية (تقريبية، مايو 2026):**

| نوع المحادثة | السعر | متى |
|-------------|-------|----|
| **Service** (يبدأها العميل) | مجانية أول 1000/شهر، ثم 0.10 ر.س | الردود التلقائية |
| **Utility** (إشعارات) | 0.10 ر.س | حالة الطلب، شحن |
| **Marketing** (حملات) | 0.20-0.30 ر.س | عروض، إعلانات |
| **Authentication** | 0.10 ر.س | OTP |

### التقدير الواقعي:
- **10,000 رسالة** ≈ **2,000 محادثة** (5 رسائل/محادثة في المتوسط)
- التوزيع الطبيعي:
  - 60% Service (1,200) → 1,000 مجانية + 200 × 0.10 = **20 ر.س**
  - 30% Utility (600) → 600 × 0.10 = **60 ر.س**
  - 10% Marketing (200) → 200 × 0.25 = **50 ر.س**
- **المجموع: ~130 ر.س**

### ⚠️ هذا أكبر من سعر الأساسية!

**الحل:**
- **رسوم Meta = على التاجر مباشرة** (مو على مشروعك)
- التاجر يضع payment method في Meta Business Manager
- مشروعك ما يدفع شي لـ Meta — هذا المهم

### 📌 المنطقة الرمادية: "Free Tier" — 1000 محادثة service مجانية
- هذي **per WhatsApp Business Account**، مو per tenant
- إذا التجار يستخدمون أرقام منفصلة → كل واحد عنده 1000 مجانية
- النموذج اللي بنيناه = كل تاجر يضع رقمه = كل تاجر عنده free tier خاص ✅

**تكلفة Meta على مشروعك = 0 ر.س** ✅

## 🎯 6. التوصيات الاستراتيجية

### ✅ التسعير الحالي ممتاز إذا طبّقت 3 حماية:

#### 1. **سياسة الاستخدام العادل** (Fair Use Policy)
أضف في الشروط:
> "غير محدود" في النمو يعني حتى **15,000 رسالة شهرياً**. بعدها تنبيه + إذا تجاوز 25,000 → 0.02 ر.س/رسالة إضافية.

كود مقترح في `planGate.js`:
```js
'النمو': {
  limits: {
    messages_monthly: 15000,         // الحد الناعم
    messages_overage_price: 0.02,    // ر.س لكل رسالة إضافية
    messages_hard_limit: 30000       // الحد الصارم
  }
}
```

#### 2. **تتبّع تكلفة كل تاجر** (Cost Tracking)
أضف عمود `actual_cost_sar` في `UsageCounter` يحسب تكلفة OpenAI الفعلية شهرياً. لما يقترب من 50% من سعر الباقة → تنبيه إداري.

#### 3. **هيكلة مختلفة للباقات الكبيرة** (Tiered Overage)
الشركات: **0.015 ر.س/رسالة** بعد 30,000 — مع فاتورة شهرية تلقائية.

### 📈 أو **رفع الأسعار** (إذا السوق يتحمل):

| الباقة الحالية | المقترح | المبرر |
|--------------|---------|--------|
| 79 ر.س | 99 ر.س | ربح أكثر، نفس الميزات |
| 149 ر.س | **199 ر.س** | يغطي خطر الاستخدام الثقيل |
| 299 ر.س | **399 ر.س** | أعلى للشركات قادرة |

> منصات مشابهة عالمياً (Tidio, Gorgias) تسعّر $39-99 = 146-371 ر.س — أسعارنا أقل.
