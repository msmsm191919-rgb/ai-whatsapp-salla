# 🚀 دليل التشغيل - نظام واتساب سلة الذكي (SaaS)

تم استكمال وتجهيز النظام بالكامل. اتبع الخطوات التالية لتشغيل النظام وتجربته.

## 1️⃣ تشغيل السيرفر (Server)
قم بفتح نافذة تيرمينال (Terminal) وشغل الأمر التالي:

```bash
npm run dev
```

> **ملاحظة:** السيرفر يعمل بشكل افتراضي على المنفذ **8090**.
> سيقوم النظام تلقائياً بإنشاء قاعدة البيانات وإدخال بيانات تجريبية (Seed).

---

## 2️⃣ تجربة النظام (Simulation)
لأننا نعمل في بيئة تطوير (Localhost)، قمنا ببناء أدوات محاكاة قوية:

### 🖥️ لوحة التحكم (Dashboard)
ادخل على الرابط التالي لرؤية واجهة التاجر وإعدادات الربط:
- 👉 [http://localhost:8090/dashboard](http://localhost:8090/dashboard)

### 💬 محاكي الواتساب (WhatsApp Simulator)
قمنا ببناء محاكي كامل للواتساب لتجربة الردود الذكية دون الحاجة لهاتف حقيقي:
- 👉 [http://localhost:8090/whatsapp-simulator](http://localhost:8090/whatsapp-simulator)

---

## 3️⃣ تجربة السيناريوهات (Scenarios)

### 🅰️ سيناريو 1: عميل يرسل رسالة (AI Chatbot)
افتح **محاكي الواتساب** في متصفحك، ثم في تيرمينال **جديد**، شغل السكربت التالي:

```bash
node simulate_incoming_msg.js "كيف اقدر استبدل المنتج؟"
```
✅ **النتيجة:** ستظهر الرسالة في المحاكي، وسيقوم الذكاء الاصطناعي بالرد عليها فوراً.

### 🅱️ سيناريو 2: طلب جديد في سلة (Order Notification)
لمحاكاة وصول طلب جديد من سلة وإرسال رسالة شكر للعميل، شغل:

```bash
node simulate_salla_webhook.js
```
✅ **النتيجة:** سيستقبل السيرفر الطلب، وسيقوم الـ AI بصياغة رسالة ترحيبية وإرسالها (ستظهر في سجلات السيرفر والداشبود).

---

## ⚙️ ملفات المشروع الرئيسية
- `app.js`: العقل المدبر للنظام (Webhooks, Express, Database).
- `aiService.js`: خدمة الذكاء الاصطناعي (OpenAI Integration).
- `views/`: واجهات المستخدم (Dashboard & Simulator).
- `helpers/ORMs/`: نماذج قاعدة البيانات.
